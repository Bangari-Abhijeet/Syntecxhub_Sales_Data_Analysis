# Syntecxhub Sales Data Analysis

## Internship Task
Data Science Internship – Week 1  
Project 2: Pandas CSV Reader & Basic Analysis

## Project Description
This project analyzes a real-world sales dataset using Python and Pandas. The goal is to explore sales data, calculate key performance indicators (KPIs), filter high-value transactions, and save processed results.

## Dataset
Real-world sales dataset containing order, customer, product, and sales information.

## Tasks Performed
- Loaded CSV data into Pandas DataFrame
- Explored dataset using head(), tail(), info()
- Generated summary statistics using describe()
- Calculated sales KPIs (average, minimum, maximum)
- Filtered high-value and shipped orders
- Selected relevant business columns
- Sliced subsets of data using iloc
- Saved filtered data to CSV

## Tools & Libraries
- Python
- Pandas
- NumPy
- Jupyter Notebook (VS Code)
